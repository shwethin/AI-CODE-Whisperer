## Phase 5: Debugging & Refinement

**Action:**
- Identified that the server wasn't restarting to pick up new routes.
- Found a stale process holding port 8000.
- **Fix:** Changed port to `8081` and enabled address reuse in `server.py`.
- Updated `server.py` to correctly serve static files.

**User Prompt:** "it takes so much to load check it"
